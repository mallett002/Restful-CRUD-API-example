# Restful-CRUD-API-example
Example making a CRUD app in Node js

## HTTP METHODS
- GET
    - Read or retrieve any resource. 
    - 200 (OK) 
    - 404 (Not found)

- POST 
    - Create a **new** resource
    - 201 (OK)
    - 404 (Not found)

- PUT
    - **Update** an existing resource
    - 200 (OK)
    - 204 (No content)
    - 404 (Not found)

- DELETE
    - Delete an existing resource
    - 200 (OK)
    - 404 (Not found)
